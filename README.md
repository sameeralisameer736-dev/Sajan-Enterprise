# Sajan Enterprise — Landing Page

**Project Overview**
- **What:** Landing page for Sajan Enterprise, a company that sells and delivers hardware products across Jammu and Kashmir.
- **Purpose:** Present the company's services, contact details, and product categories to customers and partners.

**Audience & Region**
- Customers and retailers across Jammu and Kashmir looking for hardware supplies and delivery services.

**Tech Stack**
- **Frontend:** Static HTML, CSS, and JavaScript (single-page landing site).
- **Hosting:** Works on any static web host (GitHub Pages, Netlify, simple web server).
- **Optional tooling:** Local HTTP server for development, or a build pipeline if assets or frameworks are added later.

**Project Structure**
- [index.html](index.html): Main landing page.
- [README.md](README.md): This document.
- [.gitignore](.gitignore): Git ignore rules for development.

**Run Locally**
- Open `index.html` in a browser for a quick preview.
- Recommended: serve with a local HTTP server:

```bash
python -m http.server 8000
# then open http://localhost:8000/index.html
```

**Deployment**
- Deploy to any static host. For GitHub Pages, push to a repository and enable Pages from the `main` branch root.
